# Mandelbrot Viewer

A small program to view the Mandelbrot set.

# Features

- [x] Moving and zooming around with the mouse
- [x] Coloring
- [x] Smooth coloring (almost)
- [x] Autolag is on by default

# Requirements

- OpenGL 4.6
- GLEW
- GLFW

# Installation

```bash
git clone https://github.com/Elloim/MandelbrotViewer.git
cd MandelbrotViewer
make
```
