# Mandelbrot Viewer

A small program to view the Mandelbrot set.

# Features

- [ ] Moving and zooming around with the mouse
- [ ] Coloring
- [ ] Smooth coloring

# Requirements

- OpenGL 4.6
- GLEW
- GLFW

# Installation

```bash
git clone https://github.com/Elloim/MandelbrotViewer.git
cd MandelbrotViewer
make
```
